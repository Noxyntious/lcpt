# lcpt
GTA V Linux-Windows Crossplay Toolkit
## what does this do?
it simply modifies your hosts file on windows to block battleye servers, so you can play with your linux friends again.
## why is this needed?
due to rockstar adding battleye to GTA V to attempt to remove cheaters from their game, and not enabling the linux compatibility option, battleye now blocks you from joining any server, as you get kicked within 3 minutes of playtime. this effectively creates a split between once unified windows and linux users, where windows users with battleye ON cannot play with linux users with battleye OFF. this script aims to make it easy on the windows side to hop between playing public/windows only, and playing windows/linux.
## who's fault is this?
rockstars. battleye does, and always has had opt-in linux compatibility (see Arma 3, Unturned, ARK, War Thunder), yet in GTA V, this option remains disabled, likely due to the linux option making battleye run in userspace, rather than kernel space.) although, as we know, all this did was make cheaters spend a bit more money on cheats, and linux users to have to resort to this, or kicking rocks. 
## backlash?
sure. but the truth is, rockstar just doesn't care. yes, the people that use linux (and the people that own a steam deck) were very very mad at this update, but to them, it's such a small percentage, and even less to them, those people actually gave the game money.
